# Minha Primeira IA do Zero (Keras + TensorFlow)

Este repositório contém meu primeiro projeto de inteligência artificial desenvolvido totalmente do zero, após a conclusão do ensino médio.  
O objetivo foi entender na prática como funciona o treinamento de um modelo simples em Keras: construção do dataset, treinamento, queda do loss, salvamento do modelo e previsão final.

O modelo foi treinado para aprender a relação entre valores de entrada e saída em um pequeno dataset que eu defini manualmente.  
Após 800 épocas, o loss caiu para valores muito baixos e o modelo conseguiu prever corretamente novos valores.

a saida do modelo se baseia em um valor flutuante e gera o dobro do valor de entrada menos 1
